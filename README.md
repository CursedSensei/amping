# Amping

> A gamified AI companion that supports TB treatment adherence with secure video verification.

## Overview

Amping combines **Video-Observed Therapy (VOT)**, a conversational AI companion, and gamification to help patients stay on treatment while giving healthcare teams better adherence visibility.

Core capabilities:
- Daily AI-guided check-ins (mood/symptom + motivation + dose flow)
- Automated dose recording and secure upload
- XP/streak/badge mechanics for sustained adherence
- Django-based clinical dashboard for provider review and reconciliation

## Repository Structure

- `backend/` — Django backend APIs, auth, adherence, and gamification services
- `frontend/hc_professional/` — React + Vite healthcare professional dashboard
- `frontend/patient/` — Android (Kotlin) patient application
